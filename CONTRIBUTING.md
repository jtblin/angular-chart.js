### Contributing

1. Create an issue
1. Fork the repo
1. Install dependencies: `npm install`
1. This project follows the [Google Style Guide](https://google.github.io/styleguide/jsguide.html) with TypeScript support.
1. Use `npm run dev` for a live-reload development loop.
1. Make your changes in `src/` (TypeScript) and add tests.
1. Ensure your code is lint-free: `npm run lint`. You can use `npm run lint -- --fix` to automatically fix formatting issues.
1. Run tests: `npm test` (this will run lint, typecheck, unit, and integration tests).
1. Ensure the build works: `npm run build`
1. Submit pull request
